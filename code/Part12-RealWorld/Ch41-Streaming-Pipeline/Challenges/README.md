# Ch41 챌린지

동시성·스트리밍 실전 파이프라인을 다룬다. *필수* 한 개 + *심화* 한 개.

## 필수 챌린지

### ① 여러 Part 도구의 결합

`Types/Pipeline.cs` — 한 파이프라인이 **재시도(6부 Schedule)** + **자원 개폐 보장(6부 bracket)** + **lazy 스트리밍 변환(8부)** + **동시 집계(7부 Atom)** 를 결합한다. 불안정한 연결을 재시도로 복구하고, 소스를 안전하게 개폐하며, 파싱 성공만 추려 병렬 합산한다.

**노리는 능력** — 따로 배운 도구들이 *한 실전 코드* 로 합성됨을 본다.

## 심화 챌린지 (선택)

### ② 집계 확장

`Average.cs` — 파이프라인 결과(합/개수)로 평균을 계산한다. 동시 집계 결과를 후처리하는 패턴.

**노리는 능력** — 파이프라인 출력을 조합해 새 지표를 만든다.

## 실행

dotnet run --project code/Part10-RealWorld/Ch41-Streaming-Pipeline/Ch41.csproj
