# Ch30 챌린지

효과적 lazy 스트림(StreamT)을 다룬다. *필수* 한 개 + *심화* 한 개.

## 필수 챌린지

### ① 당긴 만큼만 계산 (메모리/효과 안전)

`Program.cs` — `Tapped` 무한 스트림에 `Take(3)` 만 하면 정확히 3개만 생산된다 (당기지 않은 조각은 계산도, 부수 작용도 없음).

**노리는 능력** — 스트림이 *당김(pull) 기반 lazy 시퀀스* 라 무한/대용량도 메모리 안전함을 본다. 2부 컬렉션과의 결정적 차이.

## 심화 챌린지 (선택)

### ② 무한 소수 체

`Sieve.cs` — `From(2)` 무한 스트림에서 에라토스테네스 체를 *게으르게* 돌려 소수를 무한히 흘리되, `Take(n)` 만큼만 실제 계산한다.

**노리는 능력** — lazy 스트림으로 무한 알고리즘을 *유한하게 소비* 하는 패턴을 익힌다.
