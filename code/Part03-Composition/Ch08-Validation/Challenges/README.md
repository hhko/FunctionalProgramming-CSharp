# Ch08 챌린지

8장 §8.10 의 챌린지 3 개 정답 코드.

| 파일 | 챌린지 | 본문 자리 |
|---|---|---|
| `ApplyTrace.cs` | 챌린지 1 — `Apply` 네 분기를 손으로 따라가기 (email 만 Invalid) | §8.10 |
| `NicknameForm.cs` | 챌린지 2 — 검증자 추가 + `Curry5` + `Apply` 한 단계 더 | §8.10 |
| `BindLosesAccumulation.cs` | 챌린지 3 — `Bind` 직접 정의 시 누적이 사라지는 이유 | §8.10 |

각 챌린지 정답을 `dotnet run` 으로 직접 호출하려면 `Program.cs` 데모 절에서 각 클래스의
정적 메서드를 호출하면 됩니다. 본 폴더의 코드는 SDK 프로젝트가 자동 포함하므로 별도
참조 설정은 필요 없습니다.
